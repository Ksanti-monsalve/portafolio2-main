🌐 Portafolio Web - HTML & CSS

📌 Descripción del Proyecto

Este proyecto consiste en un portafolio web desarrollado utilizando HTML5 y CSS3. El objetivo principal fue crear una página moderna, organizada y responsive para mostrar información personal, habilidades, proyectos y medios de contacto.

El portafolio fue desplegado en Netlify para facilitar su acceso desde cualquier dispositivo mediante internet.

⸻

🚀 Tecnologías Utilizadas

* HTML5
* CSS3
* Netlify
* Visual Studio Code
* Git & GitHub

⸻
